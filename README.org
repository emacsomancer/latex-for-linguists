#+TITLE: LaTeX for Linguists
#+AUTHOR: Benjamin Slade

This is a draft of a basic "LaTeX in half an hour" guide 
aimed at linguists, but most of the material is probably
relevant for non-linguists as well.

* Instructions
- Start by opening up the [[https://raw.githubusercontent.com/emacsomancer/latex-for-linguists/refs/heads/master/basic-for-linguists.tex][main .tex file]] in a LaTeX editor (e.g., [[https://www.overleaf.com/][Overleaf]], [[https://www.xm1math.net/texmaker/][Texmaker]], [[https://www.texstudio.org/][TeXstudio]], [[https://www.gnu.org/software/auctex/][AUCTeX]]).
